## 8.8.14 Summary

Pair and triple braids arise when multiple vortex or chiral structures intertwine in a topologically constrained configuration.
Their topology is described by braid group theory and characterized by linking numbers.
Because braid topology strongly restricts structural decay, these excitations represent some of the most persistent structures supported by the CTS functional.
